# Project Running Steps

1. Build the environment from the YML file

2. Open Anaconda Prompt and run the command: `conda activate your_env_name`

3. CD to the project directory

4. Execute the command to start the server: `python manage.py runserver`

5. When the code related to `http://127.0.0.1:8000/` appears and there are no red error messages, enter this URL into your browser, and you will see the project interface.
