时序网络排名可视分析系统
摘要： 复杂网络中节点重要性的排序，在不同领域中都得到了广泛的应用。然而，
传统的网络研究仅考虑网络的拓扑结构而忽视了网络节点本身的时间因素，在节点重要
性排序的过程中仅使用当前网络的拓扑结构进行排序，因此其成熟的网络排名方法在本
质上也是静态的。然而时序网络的拓扑结构是随着时间不断进化发展的，对于其新增节
点而言，静态排名方法对其并不友好。因此，本文提出了一种时序网络排名可视分析框
架，通过构建时序窗口以及在节点和边中加入时间节点的方式改进 PageRank，以此获得
可信的节点排序分数。通过对节点和边赋予时间属性，以及改变其转移概率的方式，在
综合考虑节点时序因素的基础上，提出了一种时序权重自适应模型，以适应不同场景下
对时序网络时间特性的需求。为满足用户对比不同时序网络排序结果，追溯节点及时序
网络拓扑结构演变情况的需求，本项目设计了一套可交互的网络排序分析框架。通过对
现实数据集的实验结果分析可知，本文系统不仅能够在大多数工作场景下正常运行，还
能为用户在探索时序网络排序方面提供可靠的帮助。
关键词： 时序网络；节点排名；可视化分析
Visual Analysis System for Temporal Network Ranking
Abstract: The ranking of node importance in complex networks has been widely
applied in various fields. However, traditional network research only considers the topological
structure of the network and overlooks the temporal factor of network nodes. In the process of
ranking node importance, only the current network’s topological structure is used for sorting,
making traditional ranking methods fundamentally static. The topological structure of
temporal networks evolves over time, which is not friendly to newly added nodes using static
ranking methods. Therefore, this paper proposes a temporal network ranking visualization
framework that improves PageRank by constructing temporal windows and incorporating time
nodes into nodes and edges to obtain reliable node ranking scores. By assigning time

1 引 言
当前，复杂网络已经成为一个简单而强大的框架，用于表示和分析各种复杂系统。
复杂网络节点的重要性排名问题，对当今的现实世界有着极其广泛而深远的意义，因为
它会影响人们如何访问在线信息和产品，以及公司和政策制定者如何分配稀缺资源等。
然而，当前很多成熟的排名方法，例如 PageRank[1]等，其本质上都是静态排名方法，
即仅考虑了当前网络的拓扑结构因素对网络节点进行重要性排名。因此，当将其应用于
随着时间的推移而快速发展的时序网络时，它们表现出了很多缺点。当网络的拓扑结构
随着时间的推移不断发展变化时，对于那些新增节点而言，使用静态网络节点排序方法
对其并不友好。因为在当前的网络拓扑结构中，它们不能去吸引更多的潜在节点。例如
引文网络，如果不对文献的影响力在时间上进行约束，就会造成在文献排名时，影响力
较大的总是那些发表时间久远、被引用次数较多的文献。而新发表的文献作为在引文网
络拓扑结构中新加入的节点，几乎没有被引用的链接，故其在引文网络的排名中一直处
于比较靠后的位置，不能受到很好的重视。所以仅考虑文献之间的引用关系而忽略时序
因素对文献排名是不够的。尤其研究者们应该重视那些新发表的文献，这些文献代表着
当前的研究趋势、研究热点。
事实上，大量的研究已经致力于对在其定义方程中明确包含时间的算法进行排名。
CiteRank[2]通过指数分布的随机冲浪者将出版时间引入 PageRank 模型，而有效传染矩
阵[3]则是对经典卡茨中心性度量的改进。这些瞬态算法主要旨在抑制旧节点的优势，从
而改善最近节点获得可见性的机会。然而，这些算法都有个共同的缺点，即所展示的结
果不直观，不具有可解释性。例如，在社交网络中，不同时期的不同用户他们的活跃程
度是不同的，对于广告投资商来说，他们需要找到不同时期影响力高的用户，或者探究
不同时期的影响力变化情况，寻找演变模式，发现潜力用户。对于用户来讲，相比于排
名结果，他们更希望知道排名的原因以及未来什么因素可以使其排名上升，这在实际生
活中是非常有意义的。
本文提出了一种时序网络排名可视分析系统，通过构建时序窗口以及在节点和边中
加入时序特性的方式改进 PageRank。首先，本文构建了一个时序窗口，用户可以选择
自己感兴趣的时间点并赋予其最高权重，其结果就是在该时间段内发生事件的可见性增
大。其次，本文的系统通过对节点和边赋予时序特性，并且改变其转移概率的方式改进

2 相关工作
在这一部分，本文将从复杂网络中的排名技术、排名可视化以及时序网络可视化这
三个板块分别阐述本文的相关工作。
2. 1 排名技术
真实网络通常会随着时间的推移而演变，时间效应已被证明会强烈影响排名算法
的性质和有效性。廖等人[4]对进化复杂网络中的排序技术进行了出色的综述和调查。这
篇综述全面介绍了社会、经济和信息系统中随时间演变的基于网络的排名算法的时间
方面。在本节中，将以这项调查为基础，总结与本毕设的工作密切相关的复杂网络进化
中的排名技术。Walker 等人[5]提出了一个名为 CiteRank 的新模型，该模型在隐形传送
项中引入了对旧节点的显式时间相关惩罚。通过改进经典卡茨中心性度量，有效传染矩
阵实现了削弱旧节点的影响，提升新节点关注度的目的。Yu 等人[6]借助惩罚旧节点的
想法改进了 PageRank，并使用科学出版物引用数据表明，TimedPageRank 排名前 30 的
论文在未来几年被引用的次数比 PageRank 排名第一的论文更多。Baeza-Yates[7]也实现
了类似的显式老化因子来惩罚较老节点的想法。以往的成功工作使人们认为，通过添
加时间引物来减少甚至消除时序因素给排名带来的不合理性，来提高 PageRank 是可行
的。T-Rank[8]是 PageRank 的一个变体，其最初目的是对网站进行排名。科学文章排名算
法[9]仅使用属于某个时间窗口的链接来计算节点的分数。这两种方法启发本文以增加工
作灵活性和可用性的方式增加时序窗口。
用户也可以通过时序窗口，动态选择不同的时间段，划分出不同的时序网络进行排
名，以此用户可以实现同一节点在不同时间段内的演变追溯以及探索不同时序网络的拓
扑结构演化。

2. 3 时序网络可视化
相较于传统的静态网络，时序网络更看重网络拓扑结构随时间的演变情况。随着时
间的推移，时序网络的结构会不断发展，相应地其网络排序结果也会随之改变。对时序
网络进行可视化展示，仅靠当前流行的网络可视化手段是不行的，因为它们都是基于网
络拓扑结构保持静态不变来展开相关操作的。由于时序网络具有时序演变的特性，本文
对此进行了专门的调研。
通过对当前可视化成果的研究分析，本文总结了几种可视化时序网络的办法，分别
是使用时间轴对时序网络演变进行逐个时间点的分析和借助动画展示时序网络随时间
推移的动态演变。
Eades 和 Huang 提出了使用动画来呈现时序网络拓扑结构随时间推移的演变情况的
策略[16]，将时序网络按照时序特性划分成多个小段，按顺序将每段网络演变作为动画的
各个帧进行依次呈现。该方法可以直观地呈现出网络的演变态势，且实现较容易，但该
策略的认知成本较高。林等人[17]提出了动态影响网络（DIN），这是一种基于规则模型
表示和分析蛋白质相互作用网络的新型视觉分析技术。林等提出了一种分析时序图变化

个文献节点赋予时序特性，便于后续通过时序窗口对时序网络节点进行筛选及实现基于

计算节点转移概率。
3. 2 需求分析
通过对当前静态网络及其相关排名算法的分析，本文了解到了静态网络缺乏对节点
时序因素的考量。对于新加入的节点，由于其在短时间内不能吸引更多的潜在节点，因
而静态网络的相关排名算法对其并不友好。对于现有的时序网络排名方法而言，尽管其
对节点进行了时序特性的处理，但是并不能很好的将结果呈现出来。因此，本文认为既
要在对时序网络节点排名时综合考虑网络节点的时序特征，还要借助相关的可视化设计
来直观地呈现出时序网络的节点排名结果，为用户提供一个易于操作，能够自由探索、
比较和优化的时序网络排名可视化分析系统。通过分析总结上述问题，并对其进行迭代
式的描述和设计需求的确定，本文提炼了如下几个需求：
R1. 获得可靠的节点排序结果：由于时序网络的节点被赋予了时序特性，如何预防
时序特性对排序结果造成误差，获得可靠的排序结果是本文首要处理的问题。第二个问
题就是如何构建时序模型，使用户能够根据自己的偏好选择感兴趣的时间段，使黄金时
间段内较为活跃的节点获得更大的时序特征也就是更高的排名。因此用户必须能够选择
感兴趣的时间点，并且用户可以自己调整时间递减速度。

点在时序排名中所发生的变化并将其呈现给用户。
R3. 交互式优化和可视化反馈：时序网络排名可视化应使用户能够动态添加和删除

关重要。但是，由于这些更改可能会对排序结果产生深远的影响，因此可视化效果必须
帮助用户跟踪更改。
R4. 排序的添加和删除：随着用户推进或倒退时序窗口的时间节点，相应时序网络

6. 2 评估
相较于传统的网络排序策略，本文提出的时序网络节点重要性排名可视框架不仅充
分考虑了现实中大部分网络都具有的时序特性，设计了对新节点较为友善的时序权重自
适应模型，还为用户提供了交互式的可视化探索框架，便于用户自由且直观地对时序网
络排序结果进行分析探索。
然而，本文也存在着一些尚未解决的问题。（1）时序递减方法不够丰富。当前的系
统能够提供给用户的时序递减方法较少，不能满足绝大多数时序网络的节点排序需求，
需要在后续的工作中持续完善时序递减方法，为用户提供更多的选择。（2）不支持可视
化大型网络。本文由于使用 LineUp 技术，每次展示的排名节点数有限，不能支持节点
数众多的网络。同时由于鱼目图本身的局限性，无法对大型网络进行拓扑结构展示。在
未来本文将会继续探索新的可视化设计，克服这个缺陷。

结 论
由于当前成熟的网络排序算法往往不考虑节点的时序因素，新节点在网络中由于不
能及时吸引足够的节点而受不到足够的关注。为了提高新节点在网络中的关注度，本项
目设计了一个时序网络节点重要性排序可视框架，旨在通过为节点和边赋予时序特性并
在排序方法中添加时序递减方法来提升时序因素对网络排序的影响，同时为用户提供了
一套易于交互，直观反馈的可视化框架。依托对现实引文网络数据的分析，本毕设证实
了本项目系统在提升新节点在时序网络的关注度，以及实现时序网络节点排序方面的有
效性和可靠性。
